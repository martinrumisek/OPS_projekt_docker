# Docker - Úvod
V tomto repozitáři naleznete návody pro zprovoznění Dockeru.

## Co je to Docker?
Docker je open-source projekt umožňující izolaci aplikací do jednotlivých kontejnerů (tzv. odlehčená nebo kontejnerová virtualizace). Narozdíl od virtuálních strojů neobsahuje operační systém, ale pouze soubory potřebné pro chod aplikace. Nevýhodou je svázanost s hostujícím operačním systémem.
